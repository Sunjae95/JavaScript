# npm vs yarn
### npm 
- 장점: package 설치하는 도중에 코드 실행이 가능.
-  단점: 장점에 대한 안정성의 위험도 증가. / 정책없이 등록했던 패키지 제출물 부분에서 위험도가 높을 수 있다.

### yarn
- 장점: yarn.lock은 모든 디바이스에 같은 패키지를 설치하는 것을 보장하여 다른 버전 설치로 부터 버그가 오는 양을 많이 줄일 수 있다.
